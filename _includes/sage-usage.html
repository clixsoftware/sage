Usage

Installation

{% highlight bash %} npm install sage {% endhighlight %}

Sage exports the sage global object when used in the browser.

Synopsis

{% highlight js %} var sage = require('sage'); var es = sage(); var esi = es.index('my-index'); esi.exists(function(err, exists){ console.log(exists); }); {% endhighlight %}

Create a Sage Client

To create a Sage client, call the exported sage object with an optional URI to ElasticSearch:

{% highlight js %} // create a Sage client es = sage(); es = sage("http://localhost:9200"); es = sage("http://user:pass@localhost:9200"); // URI with index name esi = sage("http://user:pass@localhost:9200/test-index"); {% endhighlight %}
